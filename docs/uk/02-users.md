# Сторінка Users

Це головна сторінка для керування правами конкретних менеджерів. Тут ви бачите всіх менеджерів сайту та можете призначати їм ролі і точкові права.

## Таблиця менеджерів

Кожен рядок — один менеджер. Колонки:

- **ID** — ідентифікатор менеджера у системі
- **Manager** — ім'я з профілю EVO
- **Role** — поточна роль (або порожньо, якщо ще не призначена)
- **Modules** — модулі, до яких менеджер має хоч якийсь доступ
- **Grants** — смужка прогресу: скільки дій йому доступно з усіх можливих
- **Overrides** — кількість точкових дозволів/заборон поверх ролі
- **Edit** — кнопка відкриття повної матриці прав

## Відкриття матриці менеджера

Клацніть кнопку **Edit** або двічі клацніть по рядку. Відкриється попап з повною матрицею дозволів менеджера, згрупованою по модулях.

### Зміна ролі

У верхній частині попапу є випадаючий список **Role**. Виберіть нову роль — вона застосується після натискання **Save**.

### Бейджі дозволів

Кожна клітинка матриці — це **бейдж** одного з чотирьох кольорів:

- **Зелений ✓** — дія дозволена **від ролі** (успадкована)
- **Синій ✓** — дія дозволена **через override** (додана особисто цьому менеджеру)
- **Червоний ✗** — дія **заборонена через override** (знята особисто у цього менеджера, навіть якщо роль її дозволяє)
- **Сірий —** — немає доступу

### Як міняти права

Клацайте по бейджам, щоб перемикати стан:

- **Сірий → Синій** — додати індивідуальний дозвіл
- **Зелений → Червоний** — забрати дозвіл, який дає роль
- **Синій → Сірий** — прибрати доданий індивідуальний дозвіл
- **Червоний → Зелений** — повернути дозвіл ролі

Після всіх змін натисніть **Save**. Всі override-и збережуться одним пакетом.

## Акордеон модулів

У матриці модулі розгорнуті автоматично, якщо в них є хоч один дозвіл у менеджера. Модулі без доступу — згорнуті, щоб не займати простір. Клацніть на заголовок модуля, щоб розгорнути/згорнути його.

## Призначення ролі новому менеджеру

Якщо менеджер ще не має ролі, у списку він показаний з порожнім полем **Role**. Відкрийте його матрицю, виберіть роль зі списку, натисніть **Save**. Роль призначиться — менеджер отримає всі дозволи цієї ролі.
