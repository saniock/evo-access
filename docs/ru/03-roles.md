# Страница Roles

Управление ролями — создание, редактирование, клонирование, удаление.

## Что такое роль

Роль — это **именованный шаблон прав**. Вместо того чтобы вручную прописывать права каждому из 40 менеджеров, вы создаёте несколько ролей (например, `seller`, `warehouse`, `marketing`) и назначаете их группам.

Когда вы **меняете разрешения роли** (на странице *Matrix*), изменения автоматически применяются ко **всем менеджерам** с этой ролью. Это и есть главное преимущество — массовое управление.

## Создание роли

Нажмите **Create Role**. В popup заполните:

- **Name (slug)** — системное имя роли, только латинские буквы, цифры, подчёркивания. Пример: `warehouse_manager`. **Нельзя изменить** после создания.
- **Label** — отображаемое имя, может быть на вашем языке, используется в интерфейсах
- **Description** — описание назначения роли (необязательно)

После создания роль появляется в списке. Чтобы добавить ей разрешения, перейдите на страницу **Matrix**.

## Клонирование роли

Если вам нужна роль, похожая на существующую, с небольшими отличиями — не создавайте с нуля. Нажмите иконку **копирования** рядом с существующей ролью. В popup введите новое имя — система создаст новую роль со **всеми разрешениями** оригинала.

Дальше вы можете менять разрешения клона, не затрагивая оригинал.

## Редактирование

Иконка **карандаша** позволяет изменить `label` и `description`. Поле `name` (slug) изменить нельзя — оно используется в коде.

## Удаление

Иконка **корзины** удаляет роль.

- **Роль без назначенных менеджеров** — удалится сразу
- **Роль с назначенными менеджерами** — появится popup с предложением переназначить их на другую роль. Выберите новую роль — все менеджеры с текущей роли получат новую, а старая удалится. Операция атомарна: если что-то пойдёт не так, ничего не изменится.

## Системные роли

Роль `superadmin` помечена как **system** и имеет особое поведение: она **игнорирует матрицу** и даёт полный доступ ко всему. Её нельзя редактировать, клонировать или удалять через интерфейс. Используйте её только для учётных записей главных админов.
